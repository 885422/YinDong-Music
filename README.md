# YinDong-Music
一个基于 Jetpack Compose 的 Android 音乐播放器，支持多平台音乐搜索和播放
## 🎵 音动音乐 - 功能与优点
### 一、核心播放功能
功能 说明 多平台音乐搜索 支持网易云、QQ音乐、酷我音乐、酷狗音乐四大平台 在线播放 支持在线试听，无需下载 多种音质选择 标准(128kbps)、极高(320kbps)、无损(FLAC)、Hi-Res、超清母带 歌词显示 支持同步歌词、翻译歌词 播放模式 顺序播放、随机播放、单曲循环、列表循环 封面显示 圆形封面 + 环形频谱可视化效果 音频可视化 实时频谱跳动，随音乐节奏律动

### 二、搜索与发现
功能 说明 实时搜索建议 输入时自动提示相关搜索词 热搜榜单 显示热门搜索关键词 链接解析 支持抖音/汽水音乐分享链接自动解析播放 搜索历史 保存搜索记录，方便快速搜索 分类发现 抖音热歌、伤感情歌、怀旧金曲等分类

### 三、歌单管理
功能 说明 创建歌单 自定义创建个人歌单 收藏歌曲 收藏喜欢的歌曲到歌单 添加到歌单 将歌曲添加到指定歌单 播放全部 一键播放歌单所有歌曲 本地歌单 管理本地创建的歌单

### 四、下载功能
功能 说明 多音质下载 支持多种音质下载（取决于API模式） 下载管理 查看和管理已下载的歌曲

### 五、插件系统（高级功能）
功能 说明 LX Plugin 支持 支持洛雪音乐插件扩展 多源切换 可在不同音乐源之间切换 自定义API 支持自定义API地址

### 六、UI/UX 特性
功能 说明 暗色主题 精美的深色界面设计 频谱可视化 128条频谱条环绕封面，随音乐律动 封面旋转 播放时封面自动旋转 沉浸式体验 全屏歌词、沉浸式播放界面 底部导航 首页、发现、我的 三大模块

### 七、社交功能
功能 说明 官方QQ群 一键加入官方用户群（673778042） 反馈渠道 通过官方群获取支持和反馈

### 八、技术优点
优点 说明 Jetpack Compose 使用现代Android UI框架，界面流畅 Kotlin 协程 异步处理，性能优秀 多平台聚合 一个APP听遍多个平台音乐 VIP音乐支持 内置VIP Cookie，支持VIP歌曲播放 离线播放 下载后无需网络即可播放 低内存占用 优化内存使用，运行流畅 安全性 关键界面保护，防篡改机制

### 九、特色亮点
1. 🎨 精美频谱效果 - 128条紫色频谱条环绕圆形封面，随音乐实时跳动
2. 🔍 智能搜索建议 - 实时搜索建议，快速找到想听的歌曲
3. 🔗 链接解析 - 粘贴抖音/汽水音乐链接即可播放
4. 🎧 多音质支持 - 从标准音质到超清母带，满足不同需求
5. 📱 现代UI设计 - 采用Material Design 3设计规范
6. ⚡ 快速响应 - 优化的搜索和播放体验
7. 🔒 安全保护 - 关键功能完整性校验
